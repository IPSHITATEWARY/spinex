# 🌊 Spinex — AI-Powered Video Template & Editing Platform

Spinex is a modern video template and editing application built with an immersive ocean-inspired UI/UX.  
It helps creators generate aesthetic short-form video templates using AI, organize inspiration, and build custom edits with a smooth drag-and-drop workflow.

Designed for creators who make content for platforms like Instagram Reels, TikTok, and YouTube Shorts.

---

## ✨ Features

### 🎥 Video Link Parsing
- Parse Instagram & TikTok video links
- Extract metadata and media references
- Quick import workflow for inspiration-based editing

### 🤖 AI Template Generation
- Generate aesthetic video templates using AI
- Smart layout suggestions
- Auto-generated transitions and scene structures

### 🧩 Drag & Drop Template Builder
- Interactive editing experience
- Rearrange sections visually
- Modular editing blocks

### 📌 Pinterest Inspiration Integration
- Browse aesthetic inspirations
- Save moodboard references
- Generate templates inspired by trends

### 🌊 Ocean-Themed UI/UX
- Modern immersive dark interface
- Smooth animations and transitions
- Clean creator-focused workflow
